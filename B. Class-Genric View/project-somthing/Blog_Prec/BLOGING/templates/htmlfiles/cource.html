Django Commands


pip install env

pip env use to configer the details what type tools you used in your Project, In a File Format.


pipenv install python

pipenv install django

pipenv .... We use the this Command to Install the compiler for windows. Take help by offical Website of Django.


pipenv shell

We use the this Command to broow the shell in onther File or Folder. Take help by offical Website of Django.




Project Commands


django-admin startproject PROJECT_NAME

This Command use to create the Project Files.


django-admin startapp APP_NAME

This Command use to create the Project's-App Files.


python manage.py runserver

We use the this Command to Run the Project, to be sure there exist "Manage.py" file.

python manage.py createsuperuser

We use the this Command to create the Super-User (Admin).


python manage.py runserver makemigrations

we can use make migrations command to configer actual schema changement in the database.


python manage.py runserver migrate

we can use migrate command to create the actual schema in the database.
